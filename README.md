## 📝 Deskripsi
**DigiVora Matrix Explorer** adalah aplikasi berbasis Java yang dirancang untuk melakukan eksplorasi, analisis statistik, dan manipulasi data pada struktur Array Multi-dimensi (Matriks). Program ini menggabungkan output terminal yang terstruktur dengan antarmuka input dinamis menggunakan JOptionPane.

---

## 🏛️ Filosofi Nama

**DigiVora Matrix Explorer** bukan sekadar rangkaian kata, melainkan representasi dari visi teknis dan identitas pengembang:

* **DigiVora**: Gabungan dari **Digital** (ranah teknologi) dan **Voracious** (haus akan ilmu). Melambangkan semangat untuk terus "melahap" tantangan pemrograman dan menyerap teknologi baru dengan antusias.
* **Matrix**: Merepresentasikan struktur data **Array Multi-dimensi** yang menjadi inti dari program ini. Matriks melambangkan keteraturan dan logika di tengah kompleksitas data.
* **Explorer**: Mencerminkan fungsi program yang dinamis dalam menjelajahi, mencari, dan menganalisis setiap elemen data untuk memberikan wawasan yang berguna.

> *"Melahap kompleksitas data, menjelajahi setiap logika."* — **DigiVora Studio**

## 🛠️ Fitur Utama

**Matrix Visualization**: Menyajikan data array 2D secara terstruktur dalam format baris dan kolom yang rapi.
- **Statistical Analysis**: 
    - Kalkulasi otomatis nilai rata-rata (*Average*) dari seluruh elemen matriks.
    - Kalkulasi total nilai (*Sum*) khusus pada elemen dengan indeks kolom ganjil (1, 3, 5, 7).
- **Interactive Search**: Mencari koordinat spesifik `[baris, kolom]` dari nilai tertentu melalui dialog input.
- **Mass Update System**: Fitur untuk mengganti seluruh nilai yang ditemukan dengan nilai baru secara dinamis.
- **Robust Error Handling**: Dilengkapi dengan `try-catch` untuk menangani kesalahan input non-numerik.

---

## 💻 Struktur Kode

Aplikasi ini dibangun dengan prinsip **DRY (Don't Repeat Yourself)** untuk efisiensi kode:

1. **`main` Method**: Mengatur alur eksekusi program, mulai dari inisialisasi data hingga proses interaksi pengguna.
2. **`tampilkanArray` Method**: Sebuah *helper method* modular yang berfungsi untuk mencetak matriks ke console, memastikan konsistensi tampilan tanpa pengulangan kode.
